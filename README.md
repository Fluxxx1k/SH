# Secret Hitler (Консольная версия)

Это текстовая версия настольной игры «Тайный Гитлер», предназначенная для игры на одном компьютере в режиме «hot-seat».

## Как скачать игру

### Способ 1: Скачать ZIP-архив (самый простой)

Этот способ не требует установки дополнительного программного обеспечения.

1.  Перейдите на страницу проекта в GitHub.
2.  Нажмите на зелёную кнопку **Code**.
3.  В выпадающем меню выберите **Download ZIP**.
4.  После скачивания распакуйте архив:
    *   **Windows:** Щёлкните правой кнопкой мыши по файлу и выберите «Извлечь всё...».
    *   **macOS:** Дважды щёлкните по файлу.
    *   **Linux:** Распакуйте через менеджер архивов или выполните в терминале команду `unzip <имя_файла>.zip`.

### Способ 2: Использование Git

Этот способ требует установленного [Git](https://git-scm.com/).

1.  Откройте терминал (командную строку).
2.  Выполните команду:
    ```
    git clone https://github.com/Fluxxx1k/SH.git
    ```

## Как запустить игру

Для запуска игры требуется установленный Python.

1.  Откройте терминал (командную строку) в папке с игрой:
    *   **Windows:** Откройте папку с игрой, щёлкните правой кнопкой мыши по пустому месту, удерживая клавишу `Shift`, и выберите «Открыть окно PowerShell здесь» или «Открыть командную строку здесь».
    *   **macOS/Linux:** Откройте терминал, введите `cd ` (с пробелом) и перетащите папку с игрой в окно терминала, затем нажмите `Enter`.

2.  Выполните команду:
    ```
    python LAUNCHING_GAME.py
    ```

3.  Следуйте инструкциям в консоли:
    *   Игра спросит: `Do you want to play (1) or only log the game (2)?`. Введите `1` и нажмите `Enter`.
    *   Далее введите количество игроков и их имена.
    *   Для указания кол-ва игроков просто введите число от 4 до 10 (может варьироваться в зависимости от пользовательских настроек) и нажмите `Enter`.
    *   Для указания кол-ва игроков И ботов просто введите 2 числа в сумме от 4 до 10 (может варьироваться в зависимости от пользовательских настроек) через пробел и нажмите `Enter`.
    *   Для указания имени игрока просто введите его имя и нажмите `Enter`.

## Нереализованные возможности

*   **Графический интерфейс (GUI):** Игра полностью текстовая и запускается в консоли.
*   **Режим «только лог»:** Опция запуска игры в режиме «только лог» (пункт 2 при запуске) в данный момент неактивна.
*   **Сетевая игра:** Игра предназначена только для одного компьютера (режим «hot-seat»). Возможность играть по сети отсутствует.
*   **Игра с ботами:** Файл `bot.py` присутствует, но логика для игры с ботами не интегрирована в основной процесс.
